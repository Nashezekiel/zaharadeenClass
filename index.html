Here are the 3 fundamental files in frontend:

  • HTML - HyperText Markup Language. It provides the basic structure of a webpage. It defines the content such as headings, paragraphs, images, and links so the browser knows what to display.
  • CSS - Cascading Style Sheets. It controls the appearance of a webpage. It is used to add colors, layout, spacing, fonts, and overall visual style.
  • JavaScript - JS is a programming language that adds behavior and interactivity to a webpage. It allows pages to respond to user actions, update content dynamically, and create interactive features.
